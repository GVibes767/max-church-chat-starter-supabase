# MAX Church Chat Starter

This repository contains a starter kit for a chat mini‑app built for the media ministry of our Pentecostal church. The goal of this project is to provide a modern, premium code base that can be used as a MAX mini‑app to facilitate communication within the media team.

## Features

- Frontend built with React and TypeScript using Vite.
- Backend using Express and Socket.IO for real-time chat.
- Supabase integration for storage (photos/videos) and database (PostgreSQL) via Prisma.
- Role‑based access control (admin, leader, member).
- Upload and display of text, photos, and videos in the chat.
- Ready‑to‑deploy structure under `apps/web` and `apps/server`.

## Getting Started

1. Clone this repository.
2. Install dependencies for both `apps/web` and `apps/server`.
3. Create `.env` files based on the provided `.env.example` files.
4. Run the backend (`npm run dev` in `apps/server`) and the frontend (`npm run dev` in `apps/web`).

For more details see the project documentation.
